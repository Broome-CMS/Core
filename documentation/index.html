Creating Pages

To create a page simply use this in your view:

page('page-name-here');

Creating Block

Before creating a block, give NAME of page as:

controllername-methodname

To create a block simply use this in your view:

$this->widget('cms.widgets.CmsBlock', array('blockParent'=>$this->getPageData()->id, 'name'=>'BLOCK-NAME-HERE'))

Creating Content

Text Truncate

$this->widget('ext.XReadMore.XReadMore', array(
'model'=>$model,
'attribute'=>'Content',
'maxChar'=>200,
'showLink'=>false,
'linkUrl' => null,
'stripTags'=>true,
'linkText'=>'Read more...',
));

Get Site Settings

To get site settings, use this in views

gl('SETTING_NAME_HERE');

SEO

In views paste these

$this->pageTitle='my title';
$this->pageDesc='description here';
$this->pageKeywords='keywords, here, like, this';
$this->pageOgTitle = 'my title';
$this->pageOgDesc = 'my title';
$this->pageOgImage = 'my title';

Render Images

To render Images from model

$config = array(
'file' => 'files/2013-03-12/test.jpg', //only If not from model
'height' => '450',
'width' => '150',
);

$model = new CmsMedia;
$result = $model->render($config);

if insert img tag is needed instead of render use:

i('link-to-image', 'alt', array('class'=>'example'))

Creating Urls

To create Urls, paste this in views

url("/model/method", array('id'=>$id))

To Generates a hyperlink tag, go to this page for refference By Example: CHtml

l('Link Text',array('controller/action', 'param1'=>'value1'));

To create Base URL (eg: for including static stuff like images)

baseUrl()

To create url to Theme base

themeBase()

Creating ajax update link Ref

CHtml::ajaxLink( // to create button use: ajaxButton
'Update profile',
array('profile/ajaxcontent', 'id' => $id), // Yii URL
array('update' => '#my-profile') // jQuery selector
);

$this->renderPartial('ajaxView', array(), false, true);

Get Post Data and Cookies

Get data instead $_GET[], use

get('key');

Get post data instead $_POST[], use

post('key');

To auto save Form data every 10 second on local storage, use this on controller's method

Yii::app()->cms->sisyphus();

Send a Coookie

sendCookie($name,$value);

Get a Cookie

getCookie($name);

JS and CSS

To include JS file or CSS file

$cs = Yii::app()->getClientScript();
$cs->registerScriptFile('/js/yourscript.js', CClientScript::POS_END);
$cs->registerCssFile('/css/yourcss.css');

Include JS or jQuery code on page

$cs->registerScript(
'my-js-id',
'JS CODE HERE',
CClientScript::POS_END );
