Download YepTap CMS

There are a few easy ways to quickly get started with YepTap CMS.

Get Yii Framework using Composer:

Inside protected directory run: `php composer.phar install`

(For beta in Fortabitt use: `git commit --allow-empty -m 'Just run composer [trigger:composer]')`

Get database from: `/protected/data/`

Note: Make sure following directories are writable to 0777:

`/files /cache /protected/runtime`

For local environment change database config in: `/protected/config/dev.php`

For beta environment change database config in: `/protected/config/beta.php`

For live environment change database config in: `/protected/config/web.php`

Login to Admin Panel by visiting: www.yoursite.com/admin/login Login using: `admin / admin`

Pages

Managing Pages

Creating Link to Page

Copy and paste the PHP code below to your view file.


page('page-name-here');

Create Navigation of children pages

Copy and paste the PHP code below to your view file.


$this->widget('zii.widgets.CMenu',array(
    'htmlOptions'=>array('class'=>'navigation'),
    'items'=>CmsPage::createChildernMenu('aboutus'),
));

Blocks

Before creating a block, give NAME of page as: controllername-methodname

Create a new Block

Copy and paste the PHP code below to your view file.


$this->widget('CmsBlock', array('blockParent'=>$this->getPageData()->id, 'name'=>'BLOCK-NAME-HERE'))

Render Partial View

Copy and paste the PHP code below to your view file.


$this->renderPartial('PARTIAL-NAME-HERE');

Managing Content

Manage content of CMS

Text Truncate

Use this to truncate long paragraphs


$this->widget('ext.XReadMore.XReadMore', array(
    'model'=>$model,
    'attribute'=>'Content',
    'maxChar'=>200,
    'showLink'=>false,
    'linkUrl' => null,
    'stripTags'=>true,
    'linkText'=>'Read more...',
));

Get Site Settings

Copy and paste the PHP code below to your view file.


gl('SETTING_NAME_HERE');

Meta Tags

Copy and paste the PHP code below to your view file.


$this->pageTitle='my title';
$this->pageDescription='description here';
$this->pageKeywords='keywords, here, like, this';
$this->pageAuthor='Author Name';
$this->pageOgTitle = 'my title';
$this->pageOgDesc = 'my title';
$this->pageOgImage = 'my title';

Render Images

Copy and paste the PHP code below to your view file.


$config = array(
    'file' => 'files/2013-03-12/test.jpg', //only If not from model
    'height' => '450',
    'width' => '150',
    'smart_resize' => true,
);

$model = new CmsMedia;
$result = $model->render($config, true); //Set to true for dummy image

if insert img tag is needed instead of render use:


i('link-to-image', 'alt', array('class'=>'example'))

Managing URLs

Creating URLs

Copy and paste the PHP code below to your view file.


url("/model/method", array('id'=>$id))

To Generates a hyperlink tag, go to this page for refference By Example: CHtml

Copy and paste the PHP code below to your view file.


l('Link Text',array('controller/action', 'param1'=>'value1'));

To create Base URL (eg: for including static stuff like images)

Copy and paste the PHP code below to your view file.


baseUrl()

To create url to Theme base

Copy and paste the PHP code below to your view file.


themeBase()

Creating ajax update link Ref

Copy and paste the PHP code below to your view file.


CHtml::ajaxLink( // to create button use: ajaxButton
    'Update profile',
    array('profile/ajaxcontent', 'id' => $id), // Yii URL
    array('update' => '#my-profile') // jQuery selector

$this->renderPartial('ajaxView', array(), false, true);

Redirect back to last page

Copy and paste the PHP code below to your view file.


$this->redirect(lastUrl());

HTTP Requests

Get Post Data and Cookies

Get data instead $_GET[], use

Copy and paste the PHP code below to your view file.


get('key');

Get post data instead $_POST[], use

Copy and paste the PHP code below to your view file.


post('key');

To auto save Form data every 10 second on local storage, use this on controller's method

Copy and paste the PHP code below to your controller method.


Yii::app()->cms->sisyphus();

Send a Coookie

The option parameter is internally passed to the public method CHttpCookie::configure() which accepts an array with the following keys:
domain, expire, secure, path, httpOnly


sendCookie($name,$value,$cookieConfig);

Get a Cookie

Copy and paste the PHP code below to your controller method.


getCookie($name);

Delete a Cookie

Copy and paste the PHP code below to your controller method.


deleteCookie($name);

JS and CSS Styling

Working with JS and CSS

To include JS file or CSS file

Copy and paste the PHP code below to your view file.


$baseUrl = Yii::app()->theme->baseUrl;
$cs = Yii::app()->getClientScript();
$cs->registerScriptFile($baseUrl.'/js/yourscript.js', CClientScript::POS_END);
$cs->registerCssFile($baseUrl.'/css/yourcss.css');

Include JS or jQuery code on page

Copy and paste the PHP code below to your view file.


$cs->registerScript(
'my-js-id',
'JS CODE HERE',
CClientScript::POS_END );

Working With Gii

Working With Gii to auto create Model, Views and Controller for you.

To access Gii: www.yoursite.com/gii/model

DB Structured to be used:


CREATE TABLE `demo` (
    `id` int(11) unsigned NOT NULL AUTO_INCREMENT,
    `title` varchar(255) DEFAULT NULL,
    `created` timestamp NULL DEFAULT NULL,
    `updated` timestamp NULL DEFAULT NULL,
    `listing_order` int(11) DEFAULT NULL,
    `active` tinyint(4) DEFAULT NULL,
    `deleted` tinyint(4) DEFAULT NULL,
    PRIMARY KEY (`id`)
)

To show in Admin Menu, go to /config/main.php, update array adminMenu=array();
